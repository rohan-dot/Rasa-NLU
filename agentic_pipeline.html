Agent Reasoning Loop
THINK

Analyze Question Complexity

LLM determines if this is a simple single-hop question or requires multi-hop reasoning across multiple facts.

DECOMPOSE → 2-4 sub-questions if multi-hop
ACT

Semantic Search (FAISS IVF8192)

Encode question with PubMedBERT embeddings → search 37M pre-indexed vectors → return top 30 semantically similar articles in <1 second.

SEARCH_SEMANTIC(query) → cosine similarity
ACT

Keyword Search (BM25 / SQLite)

Exact keyword matching catches gene names, drug names, and rare terms that embeddings miss. Fused with semantic via Reciprocal Rank Fusion (60/40).

SEARCH_BM25(query) → term frequency scoring
OBSERVE

Evaluate Retrieved Evidence

LLM reads top passages and determines: is there enough evidence to answer this question? Checks for completeness and relevance.

EVALUATE → SUFFICIENT / INSUFFICIENT
REFLECT

Generate New Queries (if insufficient)

If evidence is insufficient, LLM generates 3 new search queries using completely different terms. Loop back to SEARCH. Up to 4 iterations.

QUERY_GEN → 3 new queries → loop back ↑
← LOOP ×4
ACT

Cross-Encoder Reranking

ms-marco-MiniLM-L-12-v2 scores each (query, passage) pair with cross-attention. 30 candidates → top 15. Under 1 second on CPU.

RERANK → cross-encoder relevance scoring
ANSWER

Multi-Pass Answer Generation

3 independent answer passes at increasing temperatures (0.15, 0.30, 0.45). Type-specific prompts for factoid, yes/no, list, summary. Adversarial yes/no reasoning.

GENERATE × 3 passes
ACT

Self-Verification

LLM re-reads evidence and checks its own answer for errors, unsupported claims, or missing key information. Corrects if needed.

VERIFY → cross-check against evidence
ACT

List Review (list questions only)

Second extraction pass: "What items did you miss?" Reviews evidence passage-by-passage to find items the first pass overlooked.

LIST_REVIEW → catch missed items
REFLECT

Consensus

Majority vote across all passes for factoid/yes-no. Union merge for list items. Median-length selection for ideal answers. Filters garbage responses.

CONSENSUS → majority vote / union merge
OUTPUT

BioASQ Submission

Documents + snippets + exact answers + ideal answers. Formatted per BioASQ spec: list-of-lists for factoid/list, string for yes/no, 200-word cap on ideal.

Phase A+ and Phase B JSON files + reasoning trace
Agent Tools

SEARCH_SEMANTIC

FAISS IVF8192 over 37M PubMedBERT embeddings. Memory-mapped, sub-second search.

SEARCH_BM25

Pre-built BM25 sparse retrieval. Catches exact terms embeddings miss.

SEARCH_HYBRID

Reciprocal Rank Fusion merging semantic (60%) + keyword (40%) results.

RERANK

Cross-encoder (ms-marco-MiniLM) scores query-passage pairs with cross-attention.

DECOMPOSE

LLM breaks multi-hop questions into independent sub-questions.

VERIFY

Self-check: LLM re-reads evidence and corrects its own answer.

SYNTHESIZE

Combines sub-answers from multi-hop decomposition into coherent final answer.
